Welcome to Google Map Maker Plus

This is a Chrome plug-in I created to speed up my Map Maker editing activity

Technical stuff

What does it do?

A few UI enhancements like:

  • Several keyboard shortcuts
  • An on screen indicator for Elastic Mode
  • Ability to set Elastic Mode on by default
  • Hiding the google search box when not in use so it doesn't get focus and steal keystrokes
