    Introduction

    This Chrome extension has been created to facilitate the use of Google Map Maker for frequent and experienced users. I created it independently from Google without any source code, non-public documentation or inside knowledge. Hopefully, the time I have spent creating this extension will be of a benefit to other mappers and ultimately users of Google Maps.

    By default, the extension comes with a long list of shortcuts. You can get the hang of using shortcuts by using a few that are general purpose or you can dive right in and create some custom shortcuts.

    Here's a few to get started with:

    enterSame as clicking on OK/Save/Done buttons and links
    .(period) Browse Intersections
    ,(comma) Browse Roads
    `(back tick) Toggle Elastic Mode
    /(forward slash) Add Restaurant

    I usually use a keyboard on my Mac with a numeric keypad so some of the default shortcuts are setup for those keys. The numbers 1 to 9 on the keypad are setup to create roads with specific attributes that I use frequently. You can edit those shortcuts to change the assigned keystrokes or attributes you use frequently or create some of your own on the Shortcuts tab. You can use modifiers like shift, option/alt and command/windows keys to create independent shortcuts, so there's lots of combinations if that's useful to you.

    Good luck and have fun. Please give me suggestions and feedback both good and bad as long as it is constructive. I can't promise I will be abel to do any specific enhancements but I'm sure there are ideas I have not thought of yet.

    You can reach me at gmmp@codeplumber.com.

    -Mike

    About GMM Plus

    Settings are synchronised between computers using your Google account if you are signed into a Google Account in Chrome, see the "Sign In" section in Setting. Changes you make in this GMM PLus options area will synchronise across all computers using the same Google account.

    This extension was created independently without the help or direction from Google, Inc. Issues undoubtedly will arise as the Google Map Maker product changes. Please discontinue use of this extension if you ever suspect any adverse affects to Google Map Maker or the map data.

    If you experience issues, please uninstall this extension and verify the issue without the extension before contacting Google. Help and/or information may be available in the Map Maker forums but since this software is produced in my spare time it may not always be functional.

    The source code is free and open and is available on GitHub for others to contribute and/or create derivative works.

    You can reset the extension to the 'factory' settings with this button. Any shortcuts you have created or customized will be lost forever. You can use Export to save a copy that can be imported later. After resetting, the list of shortcuts will be restored to the default set for this version of the extension and ny you have created or modified will be gone.

    Google Analytics is used to capture anonymous use data for this extension. The data is used to improve the extension and prioritize future development. If you choose not to contribute back to this project you may disable telemetry with the checkbox below. The data collected is very generic and NOT personally identifiable. It focuses on frequency of feature use in the extension and diagnostic data (like anomalous software behaviour and causes).
